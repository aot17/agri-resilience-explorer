# Agricultural Landscape Resilience Explorer

An exploratory geospatial project investigating agricultural resilience to climatic stress using satellite imagery and environmental datasets.

## Objectives

- Monitor vegetation dynamics
- Detect drought impacts
- Measure recovery after stress events
- Identify resilient agricultural landscapes

## Technologies

- Python
- Google Earth Engine
- GIS
- Remote Sensing
- Sentinel-2
- CHIRPS Rainfall

## Status

Project initiated June 2026.
